### 1、安装

```js
npm install ydm-antd --save
```

### 2、使用

需在项目的根节点引入 CSS 文件`import "ydm-antd/build/index.css"`
`

```js
// 完整示例
import React from "react";
import ReactDOM from "react-dom/client";
import { Button } from "ydm-antd";

import "ydm-antd/build/index.css";

function App() {
  return (
    <div>
      <Button btnType="primary">这是一个测试按钮来测试</Button>
    </div>
  );
}

const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(<App />);
```

### 3、按需加载

`ydm-antd`默认支持基 ES Modules 的 Tree Shaking , 直接引入 `import { Button } from "ydm-antd"` 就有按需加载的效果。
